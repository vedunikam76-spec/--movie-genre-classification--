# 🎬 Movie Genre Classification

## 📌 Problem Statement
Build a machine learning model that predicts the genre of a movie based on its plot summary or textual information.

## 🛠️ Approach
- Preprocessing text data using **TF-IDF** vectorization.
- Experimented with classifiers:
  - Naive Bayes
  - Logistic Regression
  - Support Vector Machines (SVM)
- Evaluated models using accuracy, precision, recall, and F1-score.

## 📊 Sample Output
### Example Input:
*"A young wizard attends a magical school and battles dark forces."*

### Predicted Genre:
**Fantasy**

### Model Performance:
- Logistic Regression Accuracy: **87%**
- Naive Bayes Accuracy: **82%**
- SVM Accuracy: **85%**
